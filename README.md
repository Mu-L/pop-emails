# Email

Send emails from your terminal.

## Command Line Interface

```bash
email --from "me@example.com" --to "you@example.com" \
    --subject "Hello, world!" --attach invoice.pdf \
    < message.md
```


## Text-based User Interface

Launch the TUI

```bash
email
```
